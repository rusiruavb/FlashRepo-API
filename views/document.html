Flashcard Documentation

About this API

Flashcards are really helpful for student and teacher to actively recall lessons that they learnt. So, this api help developer who like to implement flashcard apps with using different frontend framewroks and libraries. By using this Flashcard API developers can complete fully functional flashcard application. And also this API has end-to-end backend validations and user authentication using JWT.

Technologies

Flashcard API implements using Node JS as backend server language and Express as backend framework. To store the data Flashcard API uses MongoDB Atlas as a database service. To implement user password hashing Flashcard API uses Json Web Token (JWT). So, in the backend, all users passwords are hash into unreadable format. The important thing is hackers cannot trace users' passwords because when the passwords hashed once, it cannot convert to the plain text. So, users' passwords are securely store in the database.

API End Points

Before using any end points you have to call this url - https://apiflashcard.herokuapp.com/

Create New User Account
https://apiflashcard.herokuapp.com/user/createacc
After enter this URL, you have to give the data set like this. When you give dataset make sure to give the dataset in JSON format. And also make sure to give the exactly I mention in below example.

{
   "name": "James Wiliam",
   "email": "james@gmail.com",
   "password": "james123",
   "school": "Oxford University",
   "field": "Software Engineering",
   "profilePicture": "https://sample.com"
}

JSON Key Data Type Description
"name" String Name of the user
"email" String Email of the user
"password" String User's password
"school" String School or University of the user
"field" String Field of study
"profilePicture" String Link of the profile picture.
Don't upload the profile picture. Paste the profile image link which is available online
